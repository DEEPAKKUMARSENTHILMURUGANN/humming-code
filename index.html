IJCRT · 2025

GenLearn

AI-Based 3D Projection
Technology

PSG Polytechnic College
Coimbatore, India

Published February 2025
IJCRT2502361

www.ijcrt.org

1 / 3

IJCRT · Vol. 13, Issue 2 · Feb 2025

Abstract

GenLearn revolutionises education by combining Generative AI with immersive 3D holographic projection. It accepts text, speech, or scanned objects via NLP & ASR, and delivers personalised, interactive learning experiences.

Existing System

Static textbooks & 2D aids — limited interactivity
Lecture-based classrooms — passive absorption
E-learning modules — lack immersive experiences
One-size-fits-all — fails diverse learning styles

Proposed System

Multi-modal input — text, voice & object scan
Generative AI engine — real-time 3D model gen
Holographic display — volumetric, no glasses
Personalised ML — adapts to each user
2 / 3

System Architecture

Input Layer — Keyboard, mic (ASR) & 3D scanner
App Layer — NLP processing & Generative AI
Data Layer — Computer vision & enrichment
Output Layer — 3D holographic + voice synth

Hardware

Multi-core CPU & dedicated GPU for 3D rendering
Noise-cancelling mic & 3D scanner / camera
3D projector — real-time, no glasses needed

Software Stack

AI — TensorFlow / PyTorch
NLP — spaCy, NLTK
3D — Blender / Autodesk Maya
Frontend — React + WebGL & Three.js
DB — MongoDB / PostgreSQL
3 / 3

Implementation

Front-End — HTML/CSS, React/Vue.js, WebGL
Back-End — NLP APIs, ML services, WebSocket
AI Training — fine-tuned NLP & Gen-AI models
CV Algorithms — object recognition & 3D mapping

Testing & Deployment

Unit & Integration — component validation
UAT — real user feedback loops
Performance — load, stability & scale
Phased rollout — staging → production

Conclusion

GenLearn bridges passive education and AI-driven immersive learning — delivering multi-sensory, personalised experiences via holographic 3D projection, gesture control, and generative AI.
